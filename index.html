    Build guide

    How the chain works

    The converter handles the math. You just place, configure, and bind.

    1
    ⌑

    Choose a trigger

    Bind a Button, Pilot Seat, or Car Seat to the first Delay to start the song.

    Buttons are simple. For seats, avoid W if releasing it could trigger the song early.
    2

    Set each Delay

    Use the Property Tool to enter each shown hold time, then bind every Delay to the next Delay in its chain.

    Long waits are split into several Delays, with a maximum of 10 seconds each.
    3

    Set Music Note keys

    Configure every Music Note block, then bind each Delay to its note outputs.

    Chords use several Music Notes bound to the same Delay.
    BABFT note range

    25 playable notes, ready to place.

    Music Note blocks begin at F#3. Incrementing the key moves upward through the keyboard until F#5, then loops back. Each block can sustain for about 850 ms.
